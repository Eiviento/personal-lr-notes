# 7.3 Android BLE API 分层

## 一、对象层级

```
BluetoothAdapter — 手机蓝牙入口
  ├─ BluetoothLeScanner — 扫描器
  └─ BluetoothDevice — 远程设备代理
       └─ BluetoothGatt — 一个连接
            ├─ BluetoothGattService — GATT Service
            │   ├─ BluetoothGattCharacteristic — GATT Characteristic
            │   │   └─ BluetoothGattDescriptor — Descriptor (CCCD等)
            │   └─ ...
            └─ BluetoothGattCallback — 所有异步事件
```

## 二、核心 API

### BluetoothAdapter
isEnabled() / getBluetoothLeScanner() / getBondedDevices() / getRemoteDevice(address)

### BluetoothDevice
getAddress() / getName() / getType() / getBondState() / connectGatt() / createBond() / removeBond()

### BluetoothGatt（一个连接一个实例）
discoverServices() / readCharacteristic() / writeCharacteristic() / setCharacteristicNotification() / requestMtu() / readRemoteRssi() / requestConnectionPriority() / disconnect()+close()

### BluetoothGattCallback
onConnectionStateChange / onServicesDiscovered / onCharacteristicRead / onCharacteristicWrite / onCharacteristicChanged / onDescriptorWrite / onMtuChanged

### Characteristic
getUuid() / getProperties() / getPermissions() / getValue() / getDescriptors() / setValue()

### Descriptor
UUID_CCCD + ENABLE_NOTIFICATION_VALUE / ENABLE_INDICATION_VALUE

## 三、API ↔ 协议栈对应

| API | 协议层 |
|-----|--------|
| startScan() | LL: Standby→Scanning |
| connectGatt() | LL: Standby→Initiating→Connection |
| discoverServices() | ATT: READ_BY_GROUP_TYPE→READ_BY_TYPE→FIND_INFO |
| readCharacteristic() | ATT: ATT_READ_REQ→ATT_READ_RSP |
| writeCharacteristic() | ATT: ATT_WRITE_REQ→ATT_WRITE_RSP |
| setCharacteristicNotification()+writeDescriptor | ATT: ATT_WRITE_REQ→CCCD |
| requestMtu() | ATT: ATT_EXCHANGE_MTU_REQ/RSP |
| disconnect() | LL: LL_TERMINATE_IND |

## 四、一句话

> BluetoothAdapter=手机, BluetoothDevice=流量计, connectGatt()=建立连接, discoverServices()=发现GATT表, Characteristic/Descriptor=可读写的数据, Callback=设备通知你的唯一渠道。

## 五、生命周期

BluetoothDevice 持久 / BluetoothGatt 一个连接一个实例，close() 后失效 / Service/Char/Desc 由 Gatt 管理

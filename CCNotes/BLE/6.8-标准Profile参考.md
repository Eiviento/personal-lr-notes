# 6.8 标准 Profile 参考

## 一、Device Information Service (0x180A)

Manufacturer Name(0x2A29) / Model Number(0x2A24) / Serial Number(0x2A25) / HW Revision(0x2A27) / FW Revision(0x2A26)

App 用途：扫描列表显示设备名型号、判断固件是否需升级。

## 二、Battery Service (0x180F)

Battery Level(0x2A19), Read|Notify, 0~100%。必须配 CCCD。

## 三、Environmental Sensing (0x181A)

Temperature(0x2A6E) / Humidity(0x2A6F) / Pressure(0x2A6D)

## 四、蓝牙 5.3 流量相关

Volume Flow Rate(0x2B18): float32, m³/s
Volume Flow(0x2B1F): 累计体积

## 五、流量计自定义 Profile 设计建议

```
Device Info (0x180A)
Battery (0x180F)
   └─ Battery Level [Read|Notify] + CCCD

流量计测量服务 (128-bit自定义)
   ├─ 瞬时流速 [Read|Notify] + CCCD + CUD
   ├─ 累计流量 [Read|Notify] + CCCD + CUD
   ├─ 采样间隔 [Read|Write]
   └─ 校准参数 [Read|Write, 需加密+认证] + CUD

设备配置服务 (128-bit自定义, 可选)
   ├─ 管道内径 [Read|Write]
   ├─ 流量单位 [Read|Write]
   └─ 报警阈值 [Read|Write|Notify] + CCCD
```

### 设计要点
1. 安全分层：公开 Level1 / 业务 Level2 / 敏感 Level4
2. 所有 Notify/Indicate 必须配 CCCD
3. 标准 UUID 优先（nRF Connect 可自动解析）
4. Value 格式统一：和固件同事约定浮点格式和字节序

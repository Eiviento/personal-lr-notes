# 7.1 GAP 角色

## 一、四种 GAP 角色

| 角色 | 行为 | 可连接 |
|------|------|--------|
| Broadcaster | 只广播 (ADV_NONCONN_IND) | ❌ |
| Observer | 只扫描 | ❌ |
| Peripheral | 广播 + 可被连（做 Slave） | ✅ |
| Central | 扫描 + 可发起连（做 Master） | ✅ |

## 二、Android 固定为 Central

Android 不能做 Peripheral/Broadcaster/Observer。你的 App 站在 Central 位置：扫描流量计 → 连接 → 读写 GATT。

## 三、GAP ↔ Android API

| GAP 行为 | 链路层 | Android API |
|----------|--------|------------|
| 启动扫描 | Standby→Scanning | `startScan()` |
| 停止扫描 | Scanning→Standby | `stopScan()` |
| 发起连接 | Standby→Initiating→Connection | `connectGatt()` |
| 断开 | Connection→Standby | `disconnect()+close()` |
| RSSI | PHY 信号强度 | `ScanResult.getRssi()` / `readRemoteRssi()` |

## 四、权限版本适配

- Android 6.0~11：需要 `ACCESS_FINE_LOCATION` + GPS 开关
- Android 12+：需要 `BLUETOOTH_SCAN` + `BLUETOOTH_CONNECT`（不需要位置权限）
- Android 10-：扫描需要开位置服务（经典坑）

# 概念深度学习 · 提示词卡片

---

## 🚀 启动新主题

```
我是 [你的角色]，需要 [你的目标]，但对 [领域] 不了解。
请先了解我的背景再帮我制定学习计划。一次只问一个问题。
```

---

## 📖 学一个知识点

```
讲解 [知识点]，要求：
- 为什么需要
- 核心概念
- 图解/表格/对比
- 实际意义
先发内容别存文件，我确认后再存。
```

---

## 🔍 对比两个概念

```
详细对比 [A] 和 [B]，用表格列出每个维度的差异和适用场景。
```

---

## ❓ 追问

```
[具体概念] 中的 [具体点] 没懂，用类比帮我建立直觉。
```

---

## ⏯️ 节奏控制

```
继续。          → 下一个知识点
今天先到这。    → 暂停
```

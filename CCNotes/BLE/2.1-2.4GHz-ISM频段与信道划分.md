# 2.1 2.4GHz ISM 频段与信道划分

## 一、什么是 ISM 频段？

**ISM = Industrial, Scientific, Medical**

这是一段**不需要无线电台执照就能免费使用的无线电频段**（在全球大多数国家和地区）。ITU（国际电信联盟）划分了几段 ISM 频段：

| ISM 频段 | 频率范围 | 典型用途 |
|----------|---------|---------|
| 433 MHz | 433.05 ~ 434.79 MHz | 遥控器、TPMS（胎压监测） |
| **2.4 GHz** | **2.400 ~ 2.4835 GHz** | **BLE、Wi-Fi、Zigbee、Thread、微波炉** |
| 5.8 GHz | 5.725 ~ 5.875 GHz | Wi-Fi 5/6 的部分信道 |

**关键理解**：BLE 选择 2.4GHz ISM 频段不是因为"这里最干净"，恰恰相反——这里非常拥挤。选择它是因为：

1. **全球免费免牌照**：产品出口到任何国家都不需要单独申请频率许可
2. **天线尺寸小**：2.4GHz 的波长约 12.5cm，天线可以做得非常小（1/4 波长 ≈ 3cm），适合小型设备
3. **传播特性折中**：不像 433MHz 穿墙强但天线大，不像 5.8GHz 带宽大但穿墙差

---

## 二、BLE 的信道划分

BLE 把 2.400 ~ 2.4835 GHz 这段 83.5 MHz 宽的频谱，切成 **40 个信道**：

```
频率 (MHz)
2402 ──┬── Ch 37 (广播)     ← 2.402 GHz
       │
2404 ──┼── Ch 0  (数据)
2406 ──┼── Ch 1  (数据)
       │     ...
2426 ──┼── Ch 10 (数据)
       │
2426 ──┼── Ch 38 (广播)     ← 2.426 GHz  (刚好是 Wi-Fi Ch 6 的中心附近)
       │
2428 ──┼── Ch 11 (数据)
       │     ...
2480 ──┼── Ch 36 (数据)
       │
2480 ──┼── Ch 39 (广播)     ← 2.480 GHz
       │
2483.5 ─ 频谱上界
```

每个信道间隔：**2 MHz**

---

## 三、40 个信道的命名逻辑

| 编号范围 | 用途 | 数量 | 频率公式 |
|---------|------|------|---------|
| Ch 0 ~ 36 | **数据信道** | 37 个 | f = 2402 + n×2 MHz |
| Ch 37 | **广播信道** | 1 个 | f = 2402 MHz |
| Ch 38 | **广播信道** | 1 个 | f = 2426 MHz |
| Ch 39 | **广播信道** | 1 个 | f = 2480 MHz |

**注意**：数据信道编号 0~36，广播信道编号 37/38/39。规范里这样设计是为了在协议包中区分开来——看到信道号 ≥37 就知道这是广播信道。

---

## 四、为什么选 37/38/39 三个位置作为广播信道？

这是精心设计的结果，目的是**对抗 Wi-Fi 干扰**：

```
2.4GHz 频谱上的主要干扰源：

Wi-Fi Ch 1         Wi-Fi Ch 6         Wi-Fi Ch 11
(2412 MHz)         (2437 MHz)         (2462 MHz)
  22MHz宽              22MHz宽             22MHz宽
  ├──────┤            ├──────┤            ├──────┤
  │██████│            │██████│            │██████│
  │██████│            │██████│            │██████│
──┴──────┴────────────┴──────┴────────────┴──────┴──
2400    2412         2437   2447         2462   2484
        ↑                      ↑                 ↑
     Ch 37                  Ch 38            Ch 39
    (2402)                 (2426)           (2480)
     避开Ch1                位于Ch1和Ch6之间   避开Ch11
```

| 广播信道 | 频率 | 位置策略 |
|---------|------|---------|
| **Ch 37** | 2402 MHz | 在 Wi-Fi Ch 1 (2412) 的下方，避开其主瓣 |
| **Ch 38** | 2426 MHz | 在 Wi-Fi Ch 1 和 Ch 6 之间的缝隙 |
| **Ch 39** | 2480 MHz | 在 Wi-Fi Ch 11 (2462) 的上方，避开其主瓣 |

**为什么是 3 个而不是 1 个？**

- **冗余**：如果一个广播信道被干扰（比如靠近 Wi-Fi 路由器 Ch 6），另外两个仍然可用
- **可靠性**：设备在每个广播间隔内，可以在所有 3 个广播信道上各自发送一次广播包（或者轮转），大幅提高被发现概率
- **IEEE 802.15.2 共存**：这是蓝牙和 Wi-Fi 共存规范的一部分，3 个分散的信道 + 自适应跳频，最大化共存能力

## 五、2.4GHz 频段的"邻居们"（谁在跟你抢频谱）

```
2.400 ──────────────────────────────────────── 2.4835 GHz

BLE:   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  40 个 2MHz 信道

Wi-Fi: ██████████████████████████████████████████  3 个不重叠的 22MHz 信道
       Ch1         Ch6          Ch11
       
Zigbee/Thread: ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  16 个 2MHz 信道 (802.15.4)

微波炉:  ▒▒▒▒▒▒▒▒▒▒▒▒▒▒  (集中在 2450MHz 附近，宽度不定)

经典蓝牙: ════════════════════════════════════════  79 个 1MHz 信道
```

**实际影响**：在办公室/家庭环境中，一个 BLE 连接可能同时受到 Wi-Fi、其他 BLE 设备、Zigbee 设备甚至微波炉的干扰。这就是为什么后面要学**自适应跳频**——它能动态避开被干扰的信道。

---

## 六、对你开发的影响

| 知识点 | 实际意义 |
|--------|---------|
| 2.4GHz 是公共频段 | 流量计在工厂/实验室环境中可能受 Wi-Fi、电机等设备干扰，连接不稳定是正常的 |
| 3 个广播信道分布 | 如果设备扫描不到，可能是当前环境中 3 个广播信道都被严重干扰 |
| 信道间隔 2MHz | BLE 信道很窄，干扰通常是局部的（只影响某几个信道而非全部） |

---

## 一句话总结

> **BLE 使用 2.4GHz ISM 免执照频段，切成 40 个 2MHz 宽的信道——37 个用于连接后跳频传输数据，3 个专门用于广播发现。3 个广播信道的位置是精心避开 Wi-Fi 主要信道的结果。**
